1. 속성 선택자

스타일을 부여하고자 하는 요소 내에 작성되어있는 속성을 이용해서 요소를 선택하는 방법. 선택자 뒤에 []를 붙여서 속성과 속성값을 제시한다. 

        선택자[속성=속성값]
        선택자[속성~=속성값]
        선택자[속성|=속성값]
        선택자[속성^=속성값]
        선택자[속성$=속성값]
        선택자[속성*=속성값]

2. 자손 선택자와 후손 선택자

자손 : 바로 하위 요소들 후손 : 하위요소 전체 

        * 자손선택자(>)
        a > b {
            ...
        }

        * 후손선택자( )
        a b {
            ...
        }

3. 동위레벨 선택자

동위관계에 있는 특정요소를 선택할 때 사용(뒤에 있는 요소만) 

        * a 요소 뒤에있는 b요소 "하나만"선택
        a+b {
            ...
        }

        * a 요소 뒤에있는 "모든" b요소 선택
        a~b{
            ...
        }

4. 반응선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        * 해당 요소에 클릭이 되었을 경우 스타일을 부여 
        선택자:active{
            ...            
        }

        * 해당 요소에 마우스가 올라가는 순간 스탈을 부여 
        선택자:hover{
            ...
        }

5. 상태 선택자

요소의 상태에 따라 선택되는 선택자 

        1) checked(체크된) 상태의 요소 
        선택자:checked{
            ...
        }
    

        2) 초점(focus)이 맞춰진 요소를 선택
        선택자:focus{
            ...
        }
