FlexBox

        요소의 정렬되는 방향, 순서, 요소간의 간격을 수치적으로 처리하지 않아도 알아서 유연하게 처리해주는 형식. 

        * Flexbox의 핵심 개념
        1. Flexbox의 구성요소
         - flex-container : 정렬이 필요한 요소를 감싸는 부모 컨테이너
         - flex-item : 정렬이 적용되는 요소

        2. Flexbox의 정렬 축
         - main axis : 중심 축
         - cross axis : 교차 축
